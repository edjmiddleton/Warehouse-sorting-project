# Warehouse-sorting-project
Warehouse Management System (WMS) Project Description

This project involves the design and development of a Warehouse Management System (WMS) that simulates real-world industrial warehouse operations. The system will manage the flow of goods from suppliers to customers, ensuring accurate tracking and handling of stock within a warehouse environment.

Functionality

The system tracks all items stored in the warehouse and maintains an accurate, up-to-date view of inventory. It handles both incoming stock and outgoing stock, allowing users to clearly see what is in stock, what has left, and what remains.

User Interface

A user interface will display the current warehouse state, including item quantities and relevant item information. Users can interact with the system to add and remove stock without needing access to the source code.
